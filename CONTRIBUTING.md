# Contributing

Thanks for helping improve Repo Scout.

## Local Setup

```bash
python -m pip install -e .
python -m pip install pytest
pytest
```

## Good First Issues

Good starter contributions include:

- adding a new repository health check
- improving report wording
- adding tests for edge cases
- improving documentation

## Pull Request Guidelines

- Keep changes focused.
- Add or update tests when behavior changes.
- Include a short explanation of why the change is useful.
- Prefer clear wording over clever wording in user-facing messages.

## Maintainer Notes

Repo Scout is intentionally small and dependency-light. Before adding a dependency, please explain what complexity it removes.
